# Интернет-технологии и сетевое ПО

Добро пожаловать в репозиторий с практическими заданиями по дисциплине Интернет-технологии и сетевое ПО

В каждой папке (`weeks/week-XX`) вас ждут:
- **week-XX.md** — немного теории, чтобы погрузиться в тему.
- **questions.md** — вопросы для самопроверки.
- **starter/** — заготовка проекта, с которой можно начать.
- **tests/** — автоматические тесты, которые помогут проверить решение.

Основной язык для выполнения заданий — **Python (FastAPI)**

## Как начать

1. **Подготовка окружения**
   Убедитесь, что у вас установлен Python (версии 3.11 или выше).
   ```bash
   python -m venv .venv && source .venv/bin/activate
   pip install -r requirements.txt
   ```

2. **Экспорт данных** 
   Экспортируйте данные в переменные окружения, чтобы тесты знали, какой вариант проверять:
   ```bash
   export GROUP=431
   export STUDENT_ID=s06
   ```

3. **Запуск тестов**
   Для проверки заданий первой недели используйте команду:
   ```bash
   make test WEEK=**
   ```

## Индивидуальные задания
Вариант задания лежит в JSON-файле:
`variants/431/s06/week-XX.json`.

Вы можете быстро посмотреть свой вариант командой:
```bash
python tools/show_my_variant.py 02
```

Тесты автоматически считывают этот файл, используя ваши переменные окружения, и проверяют, соответствует ли ваше решение заданию.
