# Security Checklist (tickets-s06)

## 1. Аутентификация и доступ
- [ ] Есть ли аутентификация (JWT)?
- [ ] Проверяются ли права доступа?
- [ ] Ограничен ли доступ к чужим данным?

## 2. Валидация данных
- [ ] Проверяются ли входные данные?
- [ ] Используются ли схемы (Pydantic/Protobuf)?

## 3. Инъекции
- [ ] Есть ли защита от SQL Injection?
- [ ] Фильтруются ли пользовательские данные?

## 4. XSS / CSRF
- [ ] Экранируются ли данные?
- [ ] Есть ли защита от XSS?

## 5. Данные и безопасность
- [ ] Не хранятся ли пароли в открытом виде?
- [ ] Нет ли утечки чувствительных данных?

## 6. Сеть
- [ ] Используется ли HTTPS?
- [ ] Защищён ли трафик?

## 7. Нагрузка
- [ ] Есть ли rate limiting?
- [ ] Есть ли защита от DoS?

## 8. Логи и конфигурация
- [ ] Логи не содержат секретов?
- [ ] Нет ли ключей в коде?

## 9. Зависимости
- [ ] Обновлены ли библиотеки?
- [ ] Нет ли уязвимых пакетов?
