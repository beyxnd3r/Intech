# Вопросы для самопроверки

1. В чем разница между Forward Proxy (обычный прокси) и Reverse Proxy (обратный прокси)?
Forward Proxy
работает от имени клиента
прокси скрывает клиента от интернета


Reverse Proxy
работает от имени сервера
прокси скрывает реальные сервисы
2. Что такое Upstream в терминологии Nginx?
Upstream — это группа бэкенд-серверов, куда прокси пересылает запросы
3. Зачем скрывать реальные IP-адреса сервисов от клиента?
Для безопасноти, для защиты от DDos атак и взломов
4. Какие еще задачи, кроме маршрутизации, может выполнять API Gateway (назовите хотя бы 2)?
Аутентификация/Авторизация, ограничение частоты запросов
5. Что произойдет, если Gateway упадет? Как можно от этого защититься?
Использовать неколько образов для gateway, использовать DNS или сервисы по типу CLoudflare для безопасности и балансировки
